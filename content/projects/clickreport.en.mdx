## Overview

Click Report is a web application that lets small and medium-sized enterprises
produce sustainability reports automatically — with no prior knowledge and no
external consultants. The platform walks users through a guided questionnaire, uses
AI to turn their answers into the finished report copy, and delivers a professionally
typeset PDF that follows a recognised standard (currently VSME for SMEs).

## The challenge

Sustainability reporting is time-consuming, expensive and technically demanding for
SMEs. Requirements differ from standard to standard, the copy has to exist in several
languages, and the result still needs to look consistent and compliant. What was
needed was a solution that covered the entire process — from data collection to a
print-ready document — while staying flexible enough to add further standards without
much effort.

## The approach

I built the application as a Laravel SaaS with a Livewire frontend and a Filament
admin backend. At its core is a modular standards architecture: each sustainability
standard is a self-contained module, so new standards can be added without touching
existing code. The actual text generation runs through the Laravel AI SDK: users'
answers are passed to a language model in anonymised form, which turns them into
multilingual report copy. HTML is rendered from Blade templates and converted into a
finished PDF via Gotenberg. A particular focus was data protection — personal data is
automatically pseudonymised before every AI request.

> From the first question to the finished PDF — AI-assisted, multilingual and
> standards-compliant, without a single consultant's fee.
